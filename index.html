Atividade Grunt

Compilação LESS e compressão JavaScript

Funcionalidades

Esta página demonstra a compilação de arquivos LESS para CSS e a compressão de arquivos JavaScript usando Grunt.

Como usar

Execute os seguintes comandos no terminal:

  • npm install - Instalar dependências
  • grunt dev - Modo desenvolvimento
  • grunt production - Modo produção
